# Приложение по управлению книгами в библиотеке #

# Запуск #
1) git clone https://github.com/sagitovich/Library_App
   
Для macos:
2) virtualenv .venv
3) source .venv/bin/activate
4) python3 main.py

Для Windows:
2) python -m venv .venv
3) .venv\Scripts\activate
4) python main.py
