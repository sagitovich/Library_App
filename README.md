# Приложение по управлению книгами в библиотеке #

## Запуск ##

- git clone https://github.com/sagitovich/Library_App

Для macOS:  
- virtualenv .venv  
- source .venv/bin/activate  
- python3 main.py  

Для Windows:  
- python -m venv .venv  
- .venv\Scripts\activate  
- python main.py
